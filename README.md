# vincent-4.github.io

My personal website, hosted with GitHub Pages.

## Fork and Setup Instructions

1. Fork this repository
2. Rename it to `your-username.github.io`
3. Update `index.html`:
   - Change title and content
   - Replace `site-icon.png` in `assets/images/` (used as favicon, 32x32 or larger recommended)
   - Update [Clicky](https://clicky.com) analytics ID or remove tracking code (Clicky is a privacy-preserving alternative to Google Analytics)
4. Enable GitHub Pages:
   - Go to repository Settings â†’ Pages
   - Set source to "Deploy from a branch"
   - Select "main" branch and "/ (root)"
   - Save changes

Site will be live at `https://your-username.github.io` after ~1-2 minutes of deployment.
